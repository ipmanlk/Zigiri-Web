Zigiri Dictionary can detect your input language automatically.

Currently has over words.

API
  • Send a get request to https://www.zigiriweb.navinda.xyz/find/ YOUR_WORD .
  • It will send back the meanings as a response.
  • Response is a JSON encoded string.
  • Input language will be detected automatically.
                    
                        //send a get request to find the Sinhala definition of the English word 'rapid'.
                        $.get("https://www.zigiriweb.navinda.xyz/find/rapid", function (data, status) {
                
                            //parse json string from the server
                            let meanings = JSON.parse(data);
                            
                            //print meanings
                            for (item in meanings) {
                                console.log(meanings[item]);
                            }
                        });
